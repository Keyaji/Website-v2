Project Title

In this project we were tasked to create a conference theme for the SF Tech conference, and create the branding using that concept. This project focuses on using Augmented Reality (AR) and Virtual Reality (VR) as a tool for jobs that require hands on training. In the future this can be expanded to more specialized fields as well. The benefits for using AR and VR for training is it allows workers to train skills remotely, and Essenes the need for some tools.

Deliverables

  • Poster & Animated Poster Design
  • Badge design
  • App mock up
  • Tote bag and T-shirt branding
  • Informational kiosk
